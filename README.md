# ReadTheRoom Landing Page

Built with Next.js, Tailwind CSS, and Lucide Icons.

## Setup

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run development server:**
    ```bash
    npm run dev
    ```

3.  **Deploy:**
    Deploy to Netlify, Vercel, or any Next.js host.

## Integration with ExtensionPay

- The "Upgrade to Pro" buttons are configured to link to your ExtensionPay product.
- Ensure you have registered `read-the-room` (or your chosen ID) on ExtensionPay.com.
